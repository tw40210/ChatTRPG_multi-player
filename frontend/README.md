## Setup

- installation
  - `cd frontend`
  - `npm install`
  - `cd ../backend`
  - `python -m venv ./py_env`
  - `source ./py_env/bin/activate`
  - `pip install -r ./requirements/requirments.txt`
- lauch service
  1.  `cd frontend`
      `npm start`
  2.  `cd backend`
      `source ./py_env/bin/activate`
      `python -m src.main`
- export port to public
  - ngrok
